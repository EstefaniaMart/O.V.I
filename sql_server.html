SQL Server

Microsoft SQL Server es un sistema de gestión de bases de datos de tipo relacional desarrollado por la empresa Microsoft. SQL Server utiliza T-SQL (Transact-SQL), un lenguaje de desarrollo que implementa algunas características del lenguaje SQL estándar, utilizadas para realizar la manipulación de los datos y la creación de tablas. Cabe resaltar que este sistema gestor de bases de datos sólo se encuentra disponible para sistemas operativos Windows.

El código original de SQL Server fue desarrollo por Sybase, este código fue utilizado en la versión 4.0, posteriormente fue actualizado a las versiones 7.0 y 2000. Cada dos o tres años Microsoft lanza nuevas versiones del código ofreciendo nuevas y mejores características.

SQL Server incluye una serie de características entre ellas soporte de transacciones, soporta procedimientos almacenados, incluye un entorno gráfico de administración que permite el uso de comandos DDL y DML, permite trabajar en modo cliente-servidor, de este modo la información y los datos se alojan en el servidor y los clientes de la red sólo acceden a la información, este sistema de gestión también permite administrar información de otros servidores de datos.

Instalación de SQL Server Management Studio

Primero debe acceder al sitio web de Microsoft donde encuentra las versiones disponibles, posteriormente selecciona la versión para descargar según su sistema operativo y descarga el instalador. Para iniciar la instalación debe dar clic en Ejecutar, luego deberá elegir la opción que desea instalar. Para ver una guía de instalación más detallada puede acceder al Repositorio Institucional de la Universidad de Alicante en la siguiente URL http://hdl.handle.net/10045/25855
